# Resonance Gravity (RG)
A unified cosmological framework replacing the separate Dark Matter and Dark Energy paradigms with a single dynamical scalar field (phi). 

## Overview
Resonance Gravity is a scalar-tensor theory where the resonance field (phi) couples to the trace of the stress-energy tensor. It naturally reproduces the expansion history of the universe, provides a dynamical explanation for the Radial Acceleration Relation (RAR), and resolves the S8 tension through small-scale Jeans suppression.
